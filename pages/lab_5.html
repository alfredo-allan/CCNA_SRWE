2.0 Resolução de Erros de Conectividade Física (Links)

O validador acusou erros em Link to SW1 e Link to SW2 devido a associações incorretas de portas e tipos de mídia.

Ação Corretiva Necessária: Exclua os cabos conectados nas interfaces GigabitEthernet0/0/0 e 0/0/1 do R1. Reconecte-os utilizando o cabo correspondente exigido pela topologia original para limpar as flags vermelhas de validação.

3.0 Ajustes de Interface IPv4 em R1

Comandos executados para ativar a porta local e ajustar o IP da Serial ao escopo correto:

R1> enable
Password: class
R1# configure terminal
R1(config)# interface GigabitEthernet0/0/0
R1(config-if)# no shutdown
R1(config-if)# interface Serial0/1/0
R1(config-if)# ip address 209.165.200.225 255.255.255.252
R1(config-if)# end
R1# write memory

4.0 Remoção de Redes Duplicadas IPv6 em R2

Comando obrigatório de filtragem e limpeza de sub-redes IPv6 residuais:

R2> enable
Password: class
R2# configure terminal
R2(config)# interface GigabitEthernet0/0/1
R2(config-if)# no ipv6 address 2001:db8:c0de:14::1/64
R2(config-if)# ipv6 address 2001:db8:c0de:13::1/64
R2(config-if)# end
R2# write memory
