2.0 Ativação das Fontes de Alimentação (Modo Físico)

No modo PTPM, os ativos de rede necessitam de conexão elétrica manual para que o protocolo de linha mude para UP.

Ação Obrigatória: Conecte o cabo de força na parte traseira do switch S1 e ligue a chave de energia do roteador R1. Sem energia, os comandos no shutdown serão ignorados pelo simulador.

3.0 Ajuste das Linhas VTY e Banco Local de Usuários

Remoção de comandos de login conflitantes e reaplicação das chaves de segurança estritas de 12 caracteres:

R1> enable
Password: $cisco!PRIV*
R1# configure terminal

! Corrigindo permissões redundantes de login nas linhas virtuais
R1(config)# line vty 0 4
R1(config-line)# no login
R1(config-line)# login local
R1(config-line)# exit

! Validando os parâmetros do usuário SSH e Enable Secret
R1(config)# security password min-length 12
R1(config)# username SSHadmin secret 55Hadm!n2020
R1(config)# enable secret $cisco!PRIV*

R1(config)# end
R1# copy running-config startup-config

4.0 Ativação Definitiva das Interfaces

Após ligar os equipamentos, execute a abertura final das portas locais para sincronização dos pacotes:

R1(config)# interface GigabitEthernet0/0/0
R1(config-if)# no shutdown

R1(config)# interface GigabitEthernet0/0/1
R1(config-if)# no shutdown
R1(config-if)# end
